# DTP POC Tools

Proof-of-concept and manual analysis tools for the DTP platform.

## Tools

### [hospital-search-abbreviations](./hospital-search-abbreviations/)

Analysis tools for processing failed hospital search logs and identifying which search terms need abbreviation mappings.

**Purpose:** Improve hospital search success rates by analyzing failed searches and adding verified abbreviations.

**Status:** Active - complete workflow with 461 abbreviations deployed
